# FireSight — 60-Second Demo Script

## SETUP BEFORE RECORDING
- App open in browser, **2D Map view** active
- Fire already ignited and spreading (sim speed 10-20x so it looks active)
- Strategy panel visible on right (set to "Offensive")
- Microphone ready (V key)

---

## THE SCRIPT (60 seconds)

### [0:00–0:06] — OPEN ON LIVE FIRE (2D Map View)
**[Screen: 2D tactical map, fire already spreading. Drones orbiting. Engines moving along roads. Evacuation routes lit up.]**

> "This is FireSight. A real-time incident command system for wildfire response. What you're seeing is a Rothermel fire physics simulation running on actual terrain data — fuel models, wind, slope — with 45 autonomous agents coordinating the response."

### [0:06–0:16] — AGENTS IN ACTION (2D Map View)
**[Screen: Point at drones orbiting the fire perimeter, engines routing along roads, evacuation routes color-shifting]**

> "Each agent follows the ICS hierarchy — the same command structure real fire departments use. Recon drones are autonomously mapping the fire perimeter. Ground engines are routing on real road networks. Evacuation corridors are assessed and color-coded by threat level — all without manual input."

### [0:16–0:22] — STRATEGY CHANGE (2D Map View)
**[Action: Click "Defensive" in the strategy panel on the right. Units reposition, drones shift patterns.]**

> "The incident commander sets strategy through the ICS panel — offensive, defensive, confine. When posture changes, all 45 agents adjust in real time. Units reposition, air ops shift priority, evacuation threat levels update."

### [0:22–0:34] — COMMAND CHAIN VIEW (Press 3)
**[Action: Press 3 to switch to Command Chain view]**
**[Screen: Force-directed graph — 45 glowing nodes, message particles flowing along edges, comms log scrolling on the right]**

> "This is the full command chain. 45 agents — Command, Operations, Planning, Logistics, Finance, and five AI specialists for fire prediction, drone swarm coordination, evacuation, and resource deployment. Every message between agents is logged and visualized. Agents make proposals, the IC approves or overrides, and resources move."

**[Screen: Comms log messages scrolling — PYRO → IC, OPS → DIV-A, SWARM → DRONES]**

### [0:34–0:48] — 3D VIEW (Press 1)
**[Action: Press 1 to switch to 3D View]**
**[Screen: Fly over Google 3D Tiles terrain — fire burning with animated flames, drones with scanner beams, helicopters with water drops, tankers with retardant]**

> "The 3D view renders the same incident on real satellite terrain. Every vehicle you see — drones, helicopters, air tankers — is positioned and tasked by the agent system. You can mount any unit for a first-person view."

**[Action: Click a drone → enter FPV, fly alongside fire, HUD shows altitude/heading/battery/fire proximity]**

> "This also supports VR on PICO headsets for full spatial immersion."

### [0:48–0:57] — CLOSE (3D View, wide shot)
**[Action: ESC out of FPV, pull back to wide 3D overview with fire and all units visible]**

> "FireSight is a complete agentic mission control — voice-driven, real-time, built on real ICS protocol. 45 agents, real fire physics, three operational views, and full spatial support."

### [0:57–1:00] — END
**[Screen: Hold on the wide 3D shot or cut to title card]**

> "FireSight."

## RECORDING TIPS

1. **Fire should already be burning** when you start — don't waste time igniting on camera
2. **Sim speed 10-20x** so fire spread and unit movement are visually active
3. **Pre-set to Offensive** so the voice switch to Defensive is a visible change
4. **Practice the voice command** — one clean sentence, no hesitation
5. **Screen record at 1080p** — dark UI + fire colors pop at high res
6. **Every click deliberate** — no cursor wandering
7. **Let the IC voice play** — it sells the agentic loop (command → agent → confirmation)
