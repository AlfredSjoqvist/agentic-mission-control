I built a small experiment I’ve been calling a “mini-sized holodeck.” 🪐✨

This is a web-based AR experience powered by Gaussian splats generated from images using World Labs Marble. Each splat is ~5MB, runs smoothly in the browser, and is anchored in real time using an image target on the physical box. 📦📍

What makes this one a bit special for me:
It was built with 8th Wall, which is sunsetting next month. After building hundreds of experiences on this platform, this will likely be one of my last projects using it. ⏳ 

A nice full-circle moment—bringing together the WebAR tools I’ve built so much with, and the next generation of 3D world models. 🚀🌍

Built a off-axis projection demo using MediaPipe for face tracking, Three.js, and a 3D-scanned shoe 👟

You can try it here yourself (desktop only):
https://lnkd.in/g9GDFt3Z

For calibration, click the ⚙️ icon in the bottom left and enter your screen dimensions. After that, going full screen works best!

Visualizing my redesigned living room in 3D, all starting from a single photo.

Pipeline: Nano Banana → World Labs (Marble) → WebAR
Each of these Gaussian Splats is only ~1.5MB (.spz) 

How it came together:
1. Took a photo of my living room. 
2. Used Nano Banana to restyle it into a few different interior design directions. 
3. Uploaded those restyled images to marble.worldlabs.ai, which generated 3D Gaussian Splats from images alone. 
4. Built an interactive WebAR experience on top with Three.js.

End result: a lightweight AR walkthrough of each design variant.
